# unix_labs

Unified repository with three C++/Unix lab works from separate repositories:

- `lab1/` - content from `yungraze/lab1`
- `lab2/` - content from `yungraze/lab2`
- `lab3/` - content from `yungraze/lab3`

## Structure

```text
unix_labs/
├── lab1/
├── lab2/
└── lab3/
```

## Quick start

Use a Linux shell (or WSL/Git Bash) for `.sh` scripts and a C++ compiler for `.cpp` files.

Examples:

```bash
# Build C++ files
g++ lab2/calc.cpp -o calc
g++ lab3/hanoi.cpp -o hanoi

# Run shell scripts
bash lab1/Dev2Stg.sh
bash lab2/deploy.sh
```
